Sur le web

  • accès à l'interface d'administration: https://payonline.epfl.ch/ sous contrôle du droit payonline de l'accréditation
  • exemple de formulaire de test
    • saisir des données fictives et "Enregistrer"
    • cocher les "Conditi générales" et "Effectuer le paiement " sur le site de Postfinance
    • choisir le mode de paiement :
      • PostFinance Card : simulation automatique
      • VISA : 4111 1111 1111 1111
      • MasterCard : 5399 9999 9999 9999
    • cliquer "Payer"
    • vérifier la transaction dans la liste des transactions valides depuis la fiche de l'instance
